# Data and Metadata Redaction

[Edit this page](https://github.com/privacyguides/privacyguides.org/blob/main/docs/data-redaction.md?plain=1 "Edit this page")

Protects against the following threat(s):

- [Public Exposure](https://www.privacyguides.org/en/basics/common-threats/#limiting-public-information)

When sharing files, be sure to remove associated metadata.

## MAT2

**MAT2** is free, cross-platform software which allows you to remove metadata from various file types.

## ExifEraser (Android)

**ExifEraser** is a modern, permissionless image metadata erasing application for Android.

## Shortcuts (iOS & macOS)

You can remove image metadata with iOS/macOS shortcuts.

## ExifTool (CLI)

**ExifTool** is a Perl library and command-line application for reading, writing, and editing meta information.

## Criteria

**Please note we are not affiliated with any of the projects we recommend.**

- Apps developed for open-source operating systems must be open source.
- Apps must be free and should not include ads or other limitations.
