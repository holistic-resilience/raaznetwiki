# Security Keys

[Edit this page](https://github.com/privacyguides/privacyguides.org/blob/main/docs/security-keys.md?plain=1 "Edit this page")

Protects against the following threat(s):

- [Targeted Attacks](https://www.privacyguides.org/en/basics/common-threats/#attacks-against-specific-individuals)
- [Passive Attacks](https://www.privacyguides.org/en/basics/common-threats/#security-and-privacy)

A physical **security key** adds a very strong layer of protection to your online accounts.

## Yubico Security Key

The **Yubico Security Key** series is the most cost-effective hardware security key with FIDO Level 2 certification.

## YubiKey

The **YubiKey** series from Yubico are among the most popular security keys.

## Nitrokey

**Nitrokey** offers cost-effective FIDO2 and multi-function keys.

## Criteria

**Please note we are not affiliated with any of the projects we recommend.**

### Minimum Requirements

- Must use high-quality, tamper-resistant hardware security modules.
- Must support the latest FIDO2 specification.
- Must not allow private key extraction.
