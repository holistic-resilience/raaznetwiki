# Backdoor

When talking about computer security, a “ [backdoor ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/th/glossary/backdoor)” refers to a way to access software or hardware that is generally not visible to the user. This can have legitimate uses, like opening up troubleshooting tools, but can also be maliciously created to weaken security, or access [data ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/th/glossary/308108ca-6f5b-40fe-90ce-69fea66c19de) without a user knowing about it.
