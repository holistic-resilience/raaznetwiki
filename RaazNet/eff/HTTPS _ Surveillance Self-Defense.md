# HTTPS

If you've ever seen a web address spelled out as “ [http://www.example.com/](http://www.example.com/)”, you'll recognize the “http” bit of this term. HTTP (hypertext transfer [protocol ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/glossary/protocol)) is the way a [web browser ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/glossary/web-browser) on your machine talks to a remote web server. Unfortunately, standard http sends text insecurely across the internet. [HTTPS ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/glossary/https) (the S stands for “secure”) uses [encryption ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/glossary/encryption) to better protect the [data ![](https://ssd.eff.org/assets/efforg/info-7ae1fb29e1e734b20023f3e3470316c82b84a7e4b9f1d1e2f49e399f1f8de2d3.png)](https://ssd.eff.org/glossary/data) you send to websites, and the information they return to you, from prying eyes.
