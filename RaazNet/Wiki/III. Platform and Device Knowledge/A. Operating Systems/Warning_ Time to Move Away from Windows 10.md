# Warning: Time to Move Away from Windows 10

Microsoft is cautioning users that on **October 14, 2025**, Windows will no longer provide free software updates for Windows 10. This means elevated security risk for everyone using this operating system.

## Your Options

Users have several choices:

1. **Upgrade to Windows 11** (if hardware supports it)
2. **Pay for Extended Security Updates** through Microsoft's program, which will support updates through October 13, 2026 (buying one additional year)
3. **Switch to an alternative operating system** like Linux or ChromeOS Flex

## What You Can Do

### Option 1: Upgrade to Windows 11

Not all devices will support the newest version of Windows, but if you're running Windows 10 and your laptop supports Windows 11, [now's the time to upgrade](https://www.microsoft.com/en-us/software-download/windows11).

### Option 2: Switch to Linux

Your device can likely run other operating systems. There are meaningful advantages to free and open-source software—not only because the price is right, but because you really control how it works.

With so many Linux distributions available, getting started can feel intimidating. The key is to just begin somewhere. Much of what you learn using one Linux distribution transfers to others as well.

**Resource:** [ZDNet's guide to choosing the right Linux distribution](https://www.zdnet.com/article/how-to-choose-the-right-linux-desktop-distribution-for-you/)

### Option 3: Try ChromeOS Flex

If you're comfortable with Google and want a user-friendly experience, ChromeOS Flex is a free alternative to Windows 10 that you can install on older hardware. This can extend the life of aging computers that would otherwise be unusable.

**ChromeOS advantages:**
- [A decade of updates for recent devices](https://support.google.com/chrome/a/answer/6220366?hl=en)
- Relatively safe environment for opening suspicious files
- Good option for travel computers (easy to [wipe the device](https://support.google.com/chromebook/answer/183084?hl=en) when needed)

**Resource:** [Wired's guide to installing ChromeOS Flex](https://www.wired.com/story/how-to-install-chromeos-flex/)
