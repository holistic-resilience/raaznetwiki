# Privacy-Respecting Language Tools

Text inputted to grammar, spelling, and style checkers, as well as translation services, can contain sensitive information which may be stored on servers indefinitely and sold to third parties. The language tools listed here do not store your submitted text on a server and can be self-hosted for maximum control of your data.

## Privacy Concerns Addressed

- **Service Provider Data Collection**: These tools minimize data retention
- **Surveillance Capitalism**: Alternatives to services that monetize your writing

---

## Grammar & Spelling

### LanguageTool

**LanguageTool** is a multilingual grammar, style, and spell checker supporting more than 20 languages.

**Privacy Features:**
- According to their privacy policy, they do not store content sent for review
- Software is [self-hostable](https://dev.languagetool.org/http-server) for higher assurance

**Platform Availability:**
- Desktop: Windows, macOS
- Browser Extensions: Firefox, Chrome, Edge, Safari
- Mobile: iOS (App Store)

**Integrations:** Office suites and email clients

**Resources:**
- [Homepage](https://languagetool.org/)
- [Privacy Policy](https://languagetool.org/legal/privacy)
- [Source Code](https://github.com/languagetool-org)

## Translation Tools

### LibreTranslate

**LibreTranslate** is a free and open-source machine translation web interface and API server. It uses [Argos Translate](https://github.com/argosopentech/argos-translate) models on the backend.

**Privacy Features:**
- Available through public instances, including Tor onion services and I2P eepsites
- Can be self-hosted for maximum control over submitted text

**Resources:**
- [Homepage](https://libretranslate.com/)
- [Public Instances](https://github.com/LibreTranslate/LibreTranslate#mirrors)
- [Source Code](https://github.com/LibreTranslate/LibreTranslate)

## Selection Criteria

These tools were selected based on the following requirements:

1. **Must be open source** - Allows verification of privacy claims
2. **Must be possible to self-host** - Provides maximum data control
