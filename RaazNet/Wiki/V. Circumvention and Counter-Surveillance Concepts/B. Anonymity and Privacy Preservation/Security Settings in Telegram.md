# Security Settings in Telegram

Step-by-step guide for ensuring greater security and privacy in the Telegram messenger

---

## Introduction

The functions of various settings in the Telegram messenger to ensure security and privacy are a somewhat familiar topic for all users of this messenger. However, the details of security settings and how to correctly use them to have better privacy is a topic that is addressed step-by-step in this article.

> **Note:** The items explained in this article relate to common Telegram features in Windows, Linux, Mac, Android, and iOS versions and may differ slightly in appearance from the version you are using.

---

## 1. Managing Account Blocking

The first option in the privacy settings section is managing blocked accounts.

**Access Path:** `Settings > Privacy and Security > Blocked users`

In this section you can:

- View the list of blocked accounts and bots
- Cancel blocking by selecting **Unblock**
- Block new accounts with **Block User**

---

## 2. Managing Phone Number Visibility

One of the important features of Telegram is controlling the display of your phone number to different contacts.

**Access Path:** `Settings > Privacy and Security > Phone Number`

### Available Options:

| Option          | Description                  | Recommendation   |
| --------------- | ---------------------------- | ---------------- |
| **Everybody**   | All users see your number    | ❌ Inappropriate |
| **My contacts** | Only contacts see the number | ✅ Reliable      |
| **Nobody**      | No one sees the number       | ✅ Safe          |

### Exception Settings (Exceptions):

- **Never share with:** People or groups who should not see your number
- **Always share with:** People who should always see your number

---

## 3. Managing Last Seen Visibility

Control who sees your last activity time.

**Access Path:** `Settings > Privacy and Security > Last seen & online`

### Available Options:

| Option          | Description                      | Recommendation   |
| --------------- | -------------------------------- | ---------------- |
| **Everybody**   | Everyone sees the last seen time | ❌ Inappropriate |
| **My contacts** | Only contacts see it             | ✅ Suitable      |
| **Nobody**      | No one sees it                   | ✅ Safe          |

> **Note:** If you do not share these settings with someone, you will not see their last seen time either. Instead, an approximate time is displayed: `recently`, `within a week`, `within a month`

---

## 4. Managing Forwarded Messages

Control whether forwarded messages link to your account or not.

**Access Path:** `Settings > Privacy and Security > Forwarded messages`

### Available Options:

| Option          | Description                              | Recommendation   |
| --------------- | ---------------------------------------- | ---------------- |
| **Everybody**   | Messages link to your account            | ❌ Inappropriate |
| **My contacts** | Only messages forwarded by contacts link | ✅ Suitable      |
| **Nobody**      | No message links to your account         | ✅ Safe          |

> **Note:** By default, forwarded messages are linked to your user account.

---

## 5. Managing Profile Photo Visibility

Control who sees your profile picture.

**Access Path:** `Settings > Privacy and Security > Profile photo`

### Available Options:

| Option          | Description             | Recommendation   |
| --------------- | ----------------------- | ---------------- |
| **Everybody**   | Everyone sees the photo | ❌ Inappropriate |
| **My contacts** | Only contacts see it    | ✅ Suitable      |

---

## 6. Managing Voice Calls

Control who can make voice calls with you.

**Access Path:** `Settings > Privacy and Security > Voice calls`

### Available Options:

| Option          | Description            | Recommendation        |
| --------------- | ---------------------- | --------------------- |
| **Everybody**   | Everyone can call      | ❌ Inappropriate      |
| **My contacts** | Only contacts can call | ✅ Suitable           |
| **Nobody**      | No one can call        | ⚠️ Special conditions |

### Peer-to-Peer Settings:

> **⚠️ Important Security Note:** In direct calls (P2P), your IP address is discoverable by the other party. For greater security, set this option to **My contacts** or **Nobody** so that the connection is established through Telegram servers.

---

## 7. Managing Group Additions

Prevent unwanted addition to various groups.

**Access Path:** `Settings > Privacy and Security > Groups`

### Available Options:

| Option          | Description                     | Recommendation   |
| --------------- | ------------------------------- | ---------------- |
| **Everybody**   | Everyone can add you to a group | ❌ Inappropriate |
| **My contacts** | Only contacts can add           | ✅ Suitable      |

---

## 8. Active Sessions

Manage devices that have access to your account.

**Access Path:** `Settings > Privacy and Security > Active Sessions > Show all sessions`

### Visible Information:

- Last time of use
- Operating system type
- Geographic location
- IP address

### Actions:

- **X** next to each session: Revoke that device's access
- **Terminate All Other Sessions**: Revoke all access except the current device

> **Usage:** In case of mobile loss or suspicion of hacking, immediately cut off unauthorized access.

---

## 9. App Access Code (Passcode Lock)

Physical protection of the Telegram app with a local code.

**Access Path:** `Settings > Privacy and Security > Passcode Lock > Turn on local passcode`

### Settings:

- Set a password to access the app
- **Auto-Lock:** Auto-lock after 1 minute, 5 minutes, 1 hour, or 5 hours
- **Disable passcode:** Deactivation by entering the current code

---

## 10. Two-Step Verification

**Most important security setting** for account protection.

**Access Path:** `Settings > Privacy and Security > Two-Step Verification`

### Activation Steps:

1. Determine a secure password
2. Register a recovery email address
3. Enter the 6-digit confirmation code sent to the email

### Important Notes:

> **⚠️ Why is this setting vital?**
>
> - There is a possibility of SMS interception by security agencies
> - Hackers can access confirmation SMS messages via malware
> - Two-step password creates an extra security layer

> **⚠️ Warning:** The **Hint** option is for reminding the password, not the place to write the password itself!

> **Recommendation:** Ensure access to the recovery email account and its security.

---

## 11. Automatic Account Deletion

Determine the fate of the account in case of long-term inactivity.

**Access Path:** `Settings > Privacy and Security > Delete my account > If away for`

### Available Options:

- 1 month
- 3 months
- 6 months
- 1 year (Default)

---

## Summary of Recommended Settings

| Setting               | Recommended Option    |
| --------------------- | --------------------- |
| Phone Number          | Nobody or My contacts |
| Last Seen             | Nobody or My contacts |
| Forwarded Messages    | Nobody                |
| Profile Photo         | My contacts           |
| Voice Calls           | My contacts           |
| P2P in Calls          | Nobody or My contacts |
| Group Additions       | My contacts           |
| Two-Step Verification | **Active** ✅         |
| Local Access Code     | **Active** ✅         |
