# Warning: Time to Move Away from Windows 10

## Microsoft to Discontinue Support for Windows 10

Microsoft is cautioning users that on **October 14, 2025**, Windows will no longer provide free software updates for Windows 10. This means elevated security risk for everyone using this operating system, unless they:

- Upgrade to Windows 11, or
- Pay to receive additional security updates through Microsoft's Extended Security Updates program (supports updates through October 13, 2026)

The paid option only buys users another year before they will once again be forced to upgrade regardless.

## What You Can Do

### Option 1: Upgrade to Windows 11

Unfortunately, not all devices will support Windows 11, but if you're running Windows 10 and your laptop supports it, [now's the time to upgrade](https://www.microsoft.com/en-us/software-download/windows11).

### Option 2: Switch to Linux

Your device can likely run other operating systems. With so many versions of Linux available, people might feel intimidated about getting started, but there are meaningful advantages to free and open source software—not only because the price is right but because you really control how it works.

**Key points about Linux:**
- Getting started somewhere is the most important thing
- Don't get too hung up on choosing the "perfect" distribution
- Skills learned on one Linux distribution transfer to others

[Check out this guide](https://www.zdnet.com/article/how-to-choose-the-right-linux-desktop-distribution-for-you/) to learn more about choosing a Linux distribution.

### Option 3: Try ChromeOS Flex

If you're comfortable with Google and looking for a user-friendly experience, ChromeOS Flex may be a decent free alternative to Windows 10 that you can install on old hardware.

**Advantages of ChromeOS:**
- Can extend the life of aging hardware
- [A decade of updates for recent devices](https://support.google.com/chrome/a/answer/6220366?hl=en)
- Relatively safe environment for opening suspicious files
- Good option for a travel computer that can be [easily wiped](https://support.google.com/chromebook/answer/183084?hl=en)

[Check out Wired's guide to learn how to set it up](https://www.wired.com/story/how-to-install-chromeos-flex/).
