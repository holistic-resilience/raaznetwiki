# آموزش مدیریت کلیدهای PGP با استفاده از افزونه Mailvelope

در بخش اول آموزش استفاده از افزونه Mailvelope به شیوه ایجاد کلید PGP اشاره کردیم. در این ویدیو کوتاه، نحوه مدیریت کلید PGP (مانند افزودن نشانی ایمیل جدید، تغییر گذرواژه و...) با استفاده از افزونه Mailvelope را برای شما آماده کرده‌ایم که بخش دوم از آموزش‌های مربوط به افزونه Mailvelope است.
