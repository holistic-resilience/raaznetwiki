# Should You Try Google Advanced Protection?

Journalists and other at-risk users sometimes need stronger protections than those available in default phone settings. With Android 16, Google has expanded its Advanced Protection program to include hardened device security settings, similar to Apple's Lockdown Mode introduced in iOS 16.

## What is Advanced Protection Mode?

Google's Advanced Protection mode provides enhanced security features designed for users who face elevated digital threats. The program now includes device-level hardening in addition to its existing account protection features.

### Key Security Features

The [full list of features](https://security.googleblog.com/2025/05/advanced-protection-mobile-devices.html) is extensive, but notable protections include:

- **End-to-end encrypted security logging** to facilitate investigations of potential device breaches
- **USB connection blocking** for unknown devices when your phone is stolen or seized
- **Automatic 72-hour reboot** which re-encrypts the device
- **Technology restrictions** preventing certain features from activating in apps, web browsing, messages, calls, and network connections

For a detailed analysis, see [this write-up from the Electronic Frontier Foundation](https://www.eff.org/deeplinks/2025/06/googles-advanced-protection-arrives-android-should-you-use-it).

## What You Can Do

### Decide If It's Right for You

Advanced Protection may break certain websites and change how you use some apps, particularly messaging applications. Consider whether the enhanced security is worth these trade-offs for your situation.

### Enable Advanced Protection (Android)

**Note:** Not all Android devices have access to Android 16. The update is rolling out in phases across manufacturers, starting with Google Pixel phones.

**To enable on Pixel devices:**
1. Open the **Settings** app
2. Navigate to **Security and Privacy**
3. Select **Advanced Protection**

### Consider Lockdown Mode (iPhone)

For iPhone users, Apple's [Lockdown Mode](https://www.apple.com/newsroom/2022/07/apple-expands-commitment-to-protect-users-from-mercenary-spyware/) offers similar hardened security. It limits features known to be exploited in attacks, such as:

- Link previews in Messages
- Certain web technologies while browsing

**From experience:** Lockdown Mode does break certain websites and may block messages from contacts you haven't previously communicated with, especially Android users. Otherwise, it provides a surprisingly normal experience and represents a reasonable security compromise for many users.

**To enable Lockdown Mode:**
1. Open the **Settings** app
2. Navigate to **Privacy & Security**
3. Select **Lockdown Mode**
4. Tap **Turn On Lockdown Mode**

## Comparison: Advanced Protection vs. Lockdown Mode

| Feature | Google Advanced Protection | Apple Lockdown Mode |
|---------|---------------------------|---------------------|
| Platform | Android 16+ | iOS 16+ |
| USB blocking | Yes | Yes |
| Auto-reboot | 72 hours | Yes |
| Security logging | End-to-end encrypted | Limited |
| Web restrictions | Yes | Yes |
| Message restrictions | Yes | Yes |

## Who Should Consider These Features?

- Journalists covering sensitive topics
- Human rights activists
- Individuals facing targeted surveillance threats
- Anyone with elevated security concerns
