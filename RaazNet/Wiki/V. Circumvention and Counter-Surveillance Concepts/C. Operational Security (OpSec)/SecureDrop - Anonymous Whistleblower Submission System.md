# SecureDrop - Anonymous Whistleblower Submission System

SecureDrop is an open-source whistleblower submission system that news organizations and NGOs use to securely accept documents from anonymous sources. It is maintained by the Freedom of the Press Foundation.

## How to Submit Documents Anonymously

### Step 1: Prepare Your Environment
- Use a safe location—**never** a work computer or network
- Consider using a computer that cannot be traced back to you

### Step 2: Install Tor Browser
1. Go to [https://www.torproject.org](https://www.torproject.org/)
2. Download and install the desktop version of Tor Browser
3. Launch Tor Browser

### Step 3: Configure Security Settings
1. Click the **shield icon** in the browser toolbar
2. Click **Advanced Security Settings**
3. Select **Safest** for maximum protection
4. Close the Preferences tab

### Step 4: Access SecureDrop
- Navigate to the SecureDrop directory or the specific news organization's SecureDrop address (`.onion` URL)
- Follow the on-screen instructions to submit your documents

> **Important:** You are not anonymous while using a regular browser. Always use Tor Browser when accessing SecureDrop.

## Key Security Features

| Feature | Description |
|---------|-------------|
| **No Third Parties** | Servers are completely owned by and located within the news organization—no external parties can be secretly subpoenaed |
| **Minimizes Metadata** | Does not log IP addresses, browser information, or computer details |
| **End-to-End Encryption** | Encrypts your data both in transit and at rest |
| **Air-Gapped Architecture** | Protects against hackers through security best practices and isolated systems |
| **Free & Open Source** | Licensed as free and open source software for transparency and auditability |

## Participating Organizations

SecureDrop is used by major news organizations worldwide, including:

- **The Washington Post** (United States)
- **The Guardian** (United Kingdom)
- **Der Spiegel** (Germany)
- **The Globe and Mail** (Canada)
- **Disclose** (France)
- **Greekleaks / Reporters United** (Greece)

A complete directory of SecureDrop instances is available at [securedrop.org/directory](https://securedrop.org/directory/).
