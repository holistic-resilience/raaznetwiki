# Security in a Box Updates: First Half of 2025

*Published: June 5, 2025*

All content in the Security in a Box guides has been completely updated over the past year and a half, with continuous improvements based on community feedback. The team welcomes suggestions through [email and GitLab issues](https://gitlab.com/securityinabox/securityinabox.gitlab.io/-/blob/main/readme.md#how-to-give-feedback).

Content is now available in twelve languages: Arabic, Burmese, Chinese, Farsi, French, Indonesian, Pashto (with UN Women support), Portuguese, Russian, Spanish, Turkish, and Vietnamese.

## New Features

- **Search Engine**: Click the lens icon in the top right corner to search across all resources
- **RSS Feed**: Subscribe to blog updates via [RSS feed](https://securityinabox.org/rss.xml)
- **Assess & Plan Section**: New organizational structure including guides on:
  - [Physical security](https://securityinabox.org/en/assess-plan/physical-security)
  - [Safer communications](https://securityinabox.org/en/assess-plan/private-communication)
  - [Protecting information while traveling](https://securityinabox.org/en/assess-plan/safer-travel)

## New Guides

- **[Protect Your Information and Devices While Traveling](https://securityinabox.org/en/assess-plan/safer-travel/)** — Addresses current border security control concerns worldwide
- **[Protect Your Data When Using TikTok](https://securityinabox.org/en/tools/tiktok/)** — Privacy guidance for journalists, activists, and media outlets using the platform

## Updated Guides

- [Safer communications section](https://securityinabox.org/en/communication/) (complete overhaul)
- [Instagram privacy and protection](https://securityinabox.org/en/tools/instagram/)
- [X (Twitter) privacy and protection](https://securityinabox.org/en/tools/x-twitter/)
- [Online communication privacy](https://securityinabox.org/en/assess-plan/private-communication)

## Roadmap and Contributing
