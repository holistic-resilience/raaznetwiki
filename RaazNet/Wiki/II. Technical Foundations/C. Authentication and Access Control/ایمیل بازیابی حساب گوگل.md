# فعال‌سازی ایمیل بازیابی حساب کاربری گوگل

## معرفی

ایمیل بازیابی یکی از روش‌های مهم برای بازپس‌گیری دسترسی به حساب کاربری گوگل در صورت فراموشی رمز عبور یا مسدود شدن حساب است.

> **نکته امنیتی مهم:** برای ایمیل بازیابی، حتماً از یک آدرس ایمیل غیرعمومی و امن استفاده کنید.
