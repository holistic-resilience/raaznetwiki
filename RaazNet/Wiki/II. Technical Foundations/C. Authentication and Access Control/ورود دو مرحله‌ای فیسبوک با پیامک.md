# آموزش فعال‌سازی ورود دو مرحله‌ای فیسبوک با پیامک

## ⚠️ هشدار مهم

فعال‌سازی ورود دو مرحله‌ای با استفاده از پیامک، یکی از روش‌های **منسوخ** برای احراز هویت است. توصیه می‌شود در صورت امکان، از روش‌های امن‌تر مانند **برنامه‌های تولید رمز یک‌بار مصرف** (Authenticator Apps) استفاده کنید.
