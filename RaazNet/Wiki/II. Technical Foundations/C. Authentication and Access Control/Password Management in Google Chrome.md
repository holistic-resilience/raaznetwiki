# How to backup saved passwords in Google Chrome browser

The password manager tool in Chrome browser allows users to save their usernames and passwords and use them automatically to log in to websites. This guide explains how to import passwords from other browsers and back them up.

## Why is password management important?

Given the attacks carried out using user passwords, using a unique password for every user account is of high importance. Many users use browsers and password management tools for this purpose.

---

## Importing passwords to Google Chrome

If you were using another browser and now intend to use Google Chrome, you can import your saved information from **Internet Explorer** or **Firefox** browsers into Chrome.

### Steps to perform:

1. Enter Chrome settings
2. Select the source browser type
3. Select the **"Saved passwords"** option
4. Click on **Import**

---

## Backing up Chrome passwords

To create a backup or transfer passwords to another browser, follow the steps below:

### Steps to perform:

1. Enter the Chrome password management section
2. Select the Export option
3. A confirmation message will be displayed; click on **Export passwords**
4. In Windows 10, enter your Windows password or PIN
5. Select the destination folder to save the file

> [!warning] Security Warning
> The password backup file is saved **without encryption** and in plain text. Ensure the secure storage of this file.
