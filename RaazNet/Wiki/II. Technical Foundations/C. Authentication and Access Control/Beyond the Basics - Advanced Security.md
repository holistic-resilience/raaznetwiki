# Beyond the Basics: Advanced Digital Security

You've mastered the fundamentals of digital security and want to deepen your knowledge. This curated collection of advanced guides will help you strengthen your security practices across encryption, metadata management, software verification, and secure communications.

## Advanced Security Guides

### 1. Encryption Toolkit for Media Makers

Learn how to encrypt external storage devices including USB drives and external hard drives. Covers platform-specific tools:
- **Windows**: BitLocker To Go
- **macOS**: Disk Utility
- **Cross-platform**: VeraCrypt

### 2. Understanding Media Metadata

Metadata is "data about your data"—but what does that mean in practice? This guide explains:
- What metadata looks like and where it's stored
- How to view metadata in your files
- Techniques for controlling and removing sensitive metadata

### 3. Verifying Open Source Software

Open source software allows verification of trustworthiness. Learn how to use cryptographic signatures to confirm the legitimacy and integrity of downloaded files.

### 4. PGP with Smart Cards

Pretty Good Privacy (PGP) has encrypted communications for decades. Enhance your PGP security by storing keys on hardware smart cards like YubiKey, keeping sensitive cryptographic material isolated from your computer until needed.

### 5. Choosing a VPN

Cut through the marketing hype around Virtual Private Networks. This guide covers:
- Security considerations when evaluating VPN providers
- What VPNs can and cannot protect
- Recommendations for hardened options

### 6. Locking Down Signal

Already using Signal for encrypted messaging? Explore advanced security settings and features to minimize risk and maximize protection in your secure communications.
