# Important security tips for using the Zoom app

After people around the world were quarantined, many sought suitable tools to communicate with each other. Among them, the Zoom app was one of the programs that gained immense popularity. However, shortly after, existing flaws in this program caused concern for many. The manufacturer addressed the issues to some extent by reacting quickly and releasing security patches. However, you should know that Zoom company updates do not fix all the problems of this program.

---

## Securing the user account

When setting up your account, you must implement basic protection principles:

- Using a **strong and unique password**
- Enabling **two-step verification**

### Protecting the Personal Meeting ID (PMI)

After creating an account on Zoom, in addition to a username and password, you have a **personal ID for entering meetings**. Avoid making this ID public. If someone has your specific meeting ID, they can join calls hosted by you.

---

## Using work email

One of Zoom's flaws is that it considers emails with unknown domains as belonging to one company and shares their contact information with other users of the same domain.

**Recommendations:**

- Use your work email
- If you do not have a work email, use an email with a known domain (like Gmail or Yahoo)

---

## Beware of fake apps

The number of malicious files named after video calling services including Zoom, Cisco Webex, and GoToMeeting has increased drastically. Attackers abuse the popularity of these apps to distribute their malware with the appearance of these apps.

**Safe sources for download:**

- [Official Zoom website](https://zoom.us) for Windows and Mac
- App Store for iOS
- Google Play Store for Android

---

## Meeting security

### Not sharing links on social networks

Even if meetings are public, you should not share the link to join it on social networks. Hackers can exploit Zoom app vulnerabilities to steal your information.

### Password protection

Setting a password for calls is the best solution to ensure that only intended individuals can attend the meeting. Zoom has now automatically enabled passwords for all calls.

> **Note:** Meeting passwords, like links, should never be shared on social networks or public channels.

### Using the Waiting Room

This feature keeps participants in a waiting room until the host approves their entry. Benefits of this feature:

- Controlling people who enter meetings
- Ability to remove unwanted people from the meeting

---

## Screen sharing

Pay attention to the following points when using this feature:

- You can restrict sharing to the host
- It is possible to enable it for all participants
- Close all unnecessary windows before sharing

---

## Using the web version

Zoom apps on various operating systems have different flaws. Some versions allow attackers to access the device's camera and microphone.

**Advantages of the web version:**

- Based on browser sandbox
- Needs fewer permissions
- Reduced potential damages

---

## About end-to-end encryption

Zoom's reputation was partly due to advertising end-to-end encryption. However, security researchers' investigations show that **this claim is not entirely true**. In true end-to-end encryption, all conversations can only be decrypted by the communicating parties.

---

## General privacy tips

This point applies to all video conferencing services:

- Before making a call, think about what others can hear or see
- Check your surroundings
- Close unnecessary windows when sharing the screen
