# Should You Try Google Advanced Protection?

Journalists and other at-risk users sometimes need stronger protections than those available in default phone settings. Both Google and Apple now offer hardened security modes designed to limit attack surfaces by restricting device features commonly exploited in targeted attacks.

## Android 16: Advanced Protection Mode

With the release of Android 16, Google has expanded its long-standing Advanced Protection program to include hardened device security settings. This brings Android closer to feature parity with Apple's Lockdown Mode, which has been available since iOS 16 in 2022.

### Key Features of Advanced Protection

Google's Advanced Protection mode includes [numerous security enhancements](https://security.googleblog.com/2025/05/advanced-protection-mobile-devices.html). Notable features include:

- **End-to-end encrypted security logging** — Facilitates investigations of potential device breaches
- **USB connection blocking** — Prevents unknown USB connections when the device is locked
- **Automatic reboot after 72 hours** — Re-encrypts the device if stolen or seized
- **Technology restrictions** — Blocks certain potentially risky technologies in apps, web browsing, messages, calls, and network connections

For a detailed analysis, see [this write-up from Thorin Klosowski at the Electronic Frontier Foundation](https://www.eff.org/deeplinks/2025/06/googles-advanced-protection-arrives-android-should-you-use-it).

## Apple's Lockdown Mode

Since iOS 16, Apple has offered [Lockdown Mode](https://support.apple.com/en-us/105120), which limits security risks by:

- Suspending certain device features known to be exploited in previous attacks
- Disabling link previews in Messages
- Restricting certain web technologies when browsing

## What You Can Do

### Decide If It's Right for You

Advanced Protection and Lockdown Mode may break certain websites and change how you use some apps, particularly messaging applications. However, this trade-off may be worthwhile for the enhanced security these modes provide.

### Enable Advanced Protection (Android)

**Note:** Not all Android devices have access to Android 16. The update is being released as a phased rollout across manufacturers, starting with Google Pixel phones.

**To enable Advanced Protection:**
1. Open the **Settings** app
2. Navigate to **Security and Privacy**
3. Select **Advanced Protection**
4. Follow the prompts to enable

### Enable Lockdown Mode (iPhone)

[Learn more about Lockdown Mode](https://www.apple.com/newsroom/2022/07/apple-expands-commitment-to-protect-users-from-mercenary-spyware/) before deciding. From practical experience, it does break certain websites and may block messages from contacts you haven't previously communicated with, particularly those using Android. Otherwise, it provides a surprisingly normal experience.

**To enable Lockdown Mode:**
1. Open the **Settings** app
2. Navigate to **Privacy & Security**
3. Select **Lockdown Mode**
4. Tap **Turn On Lockdown Mode**
