# SecureDrop - Anonymous Whistleblower Submission System

SecureDrop is an open-source whistleblower submission system that media organizations and NGOs can install to securely accept documents from anonymous sources. It was originally designed by the late Aaron Swartz and is now maintained by the Freedom of the Press Foundation.

## How to Submit Documents Anonymously

### Step 1: Prepare Your Environment

- **Use a safe location** — Never use a work computer or network
- **Use a trusted device** — Avoid computers that may be monitored

### Step 2: Install Tor Browser

1. Go to [https://www.torproject.org](https://www.torproject.org/)
2. Download and install Tor Browser for your operating system
3. Launch Tor Browser

### Step 3: Configure Security Settings

1. Click the **shield icon** in the browser toolbar
2. Click **Advanced Security Settings**
3. Select **Safest** for maximum protection
4. Close the Preferences tab

### Step 4: Access SecureDrop

Navigate to the SecureDrop instance of your chosen news organization through the Tor network. You are now anonymous online.

## Key Security Features

### No Third Parties
The server is completely owned by and physically located inside the news organization. No external parties can be secretly subpoenaed for your information.

### Minimal Metadata Collection
SecureDrop does not log:
- Your IP address
- Browser information
- Computer details

### End-to-End Encryption
All data is encrypted both in transit and at rest, ensuring your submissions remain protected.

### Protection Against Hackers
The system enforces security best practices for journalists and is designed for use in high-risk environments.

### Free and Open Source
SecureDrop is licensed as free and open source software, allowing independent security audits and transparency.
