# بررسی فایل‌های آلوده با استفاده از سرویس ویروس‌توتال

با استفاده از سرویس **ویروس‌توتال** می‌توانید از آلودگی یا صحت و سلامت فایل‌های ناشناس و مشکوک اطلاع کسب کنید.
