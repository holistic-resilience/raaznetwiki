# Privacy-Respecting Language Tools

Text submitted to grammar checkers, spelling tools, and translation services can contain sensitive information. Many commercial services store your submitted text on their servers indefinitely and may sell it to third parties. The tools listed here prioritize privacy by not storing your submitted text and offering self-hosting options for maximum data control.

## Grammar & Spelling

### LanguageTool

**LanguageTool** is a multilingual grammar, style, and spell checker supporting more than 20 languages.

**Privacy Features:**
- According to their privacy policy, they do not store content sent for review
- Can be [self-hosted](https://dev.languagetool.org/http-server) for higher assurance

**Availability:**
- Desktop: Windows, macOS
- Browser extensions: Firefox, Chrome, Edge, Safari
- Mobile: iOS (App Store)
- Integrations available for various [office suites](https://languagetool.org/services#text_editors) and [email clients](https://languagetool.org/services#mail_clients)

**Links:** [Homepage](https://languagetool.org/) | [Privacy Policy](https://languagetool.org/legal/privacy) | [Documentation](https://languagetooler.freshdesk.com/en/support/solutions) | [Source Code](https://github.com/languagetool-org)

## Translation Tools

### LibreTranslate

**LibreTranslate** is a free and open-source machine translation web interface and API server. It uses [Argos Translate](https://github.com/argosopentech/argos-translate) models on the backend.

**Privacy Features:**
- Fully open-source
- Can be self-hosted for complete control over submitted text
- Available through public instances, including Tor onion services and I2P eepsites

**Links:** [Homepage](https://libretranslate.com/) | [Public Instances](https://github.com/LibreTranslate/LibreTranslate#mirrors) | [Source Code](https://github.com/LibreTranslate/LibreTranslate)

## Selection Criteria

These tools were selected based on the following requirements:

- **Open source** - Code must be publicly available for review
- **Self-hostable** - Must be possible to run on your own infrastructure
