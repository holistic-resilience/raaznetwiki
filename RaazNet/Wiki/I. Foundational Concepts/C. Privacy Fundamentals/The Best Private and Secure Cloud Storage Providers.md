# Private and Secure Cloud Storage Providers

Many cloud storage providers require your full trust that they will not access your files. The alternatives listed below eliminate the need for trust by implementing secure **end-to-end encryption (E2EE)**.

If these alternatives do not fit your needs, consider using encryption software like [Cryptomator](https://www.privacyguides.org/en/encryption/#cryptomator-cloud) with another cloud provider. Using Cryptomator in conjunction with any cloud provider may be a good idea to reduce the risk of encryption flaws in a provider's native clients.

> **Note on Nextcloud:** For technical users, Nextcloud remains a recommended tool for self-hosting a file management suite. However, third-party Nextcloud storage providers are not currently recommended due to concerns about Nextcloud's built-in E2EE functionality for home users.

## Recommended Providers

### Proton Drive

**Proton Drive** is an encrypted cloud storage provider from the creators of Proton Mail.

**Key Features:**
- End-to-end encryption
- Initial free storage: 2 GB (expandable to 5 GB)
- Available on: Android, iOS, Windows, macOS, Web

**Security Audits:**
- Web application independently audited by Securitum (2021)
- Mobile clients have not yet been publicly audited

**Resources:** [Homepage](https://proton.me/drive) | [Privacy Policy](https://proton.me/drive/privacy-policy) | [Documentation](https://proton.me/support/drive) | [Source Code](https://github.com/ProtonMail/WebClients)

---

### Tresorit

**Tresorit** is a Swiss-Hungarian encrypted cloud storage provider founded in 2011, now owned by Swiss Post.

**Key Features:**
- End-to-end encryption
- Swiss jurisdiction
- Available on: Android, iOS, Windows, macOS, Linux, Web

**Security Certifications & Audits:**
- **2022:** ISO/IEC 27001:2013 Compliance Certification by TÜV Rheinland InterCert Kft
- **2021:** Penetration Testing by Computest
- **2019:** Penetration Testing by Ernst & Young (confirmed data confidentiality claims)
- **Digital Trust Label** from the Swiss Digital Initiative (35 criteria covering security, privacy, and reliability)

**Resources:** [Homepage](https://tresorit.com/) | [Privacy Policy](https://tresorit.com/legal/privacy-policy) | [Documentation](https://support.tresorit.com/)

---

### Peergos

**Peergos** is a decentralized, open-source platform for storage, social media, and applications.

**Key Features:**
- Quantum-resistant end-to-end encryption
- Decentralized architecture built on IPFS (InterPlanetary File System)
- Self-hostable
- Sync engine for local folder synchronization
- WebDAV bridge for third-party application access
- Available on: Android, Windows, macOS, Linux, Web

**Security Audits:**
- **November 2024:** Audited by Radically Open Security (all issues fixed)
- **June 2019:** Audited by Cure53 (all issues fixed)

**Resources:** [Homepage](https://peergos.org/) | [Privacy Policy](https://peergos.net/privacy.html) | [Documentation](https://book.peergos.org/) | [Source Code](https://github.com/Peergos/Peergos)

## Selection Criteria

### Minimum Requirements

All recommended providers must meet these criteria:

- **Enforce end-to-end encryption**
- Offer a free plan or trial period for testing
- Support TOTP, FIDO2 multifactor authentication, or passkey logins
- Provide a web interface with basic file management functionality
- Allow easy export of all files and documents

### Best-Case Criteria

Ideal providers should also offer:

- Open-source clients
- Independent third-party audits of all clients
- Native clients for Linux, Android, Windows, macOS, and iOS
- Easy file sharing with other users
- Basic file preview and editing on the web interface

## Additional Notes

**ISO/IEC 27001:2013** compliance relates to a company's information security management system and covers the sales, development, maintenance, and support of cloud services.
