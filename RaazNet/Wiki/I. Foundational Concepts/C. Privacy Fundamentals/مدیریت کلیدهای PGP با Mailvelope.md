# آموزش مدیریت کلیدهای PGP با استفاده از افزونه Mailvelope

## مقدمه

در بخش اول آموزش استفاده از افزونه Mailvelope به شیوه ایجاد کلید PGP اشاره کردیم. در این آموزش، نحوه مدیریت کلید PGP را با استفاده از افزونه Mailvelope بررسی می‌کنیم.

### قابلیت‌های مدیریت کلید

- افزودن نشانی ایمیل جدید به کلید موجود
- تغییر گذرواژه کلید
- صادر کردن و وارد کردن کلیدها
- مدیریت کلیدهای عمومی مخاطبین
