# Guide on installing and using the uBlock Origin extension in Firefox browser

## Introduction

The **uBlock Origin** extension is a free and open-source tool for web browsers that has two main functions:

1. **Removing annoying ads** on websites while browsing
2. **Preventing tracking** by websites and advertising companies
