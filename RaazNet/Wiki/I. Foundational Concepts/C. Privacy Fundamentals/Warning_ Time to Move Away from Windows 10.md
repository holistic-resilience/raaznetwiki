# Warning: Time to Move Away from Windows 10

## Microsoft to Discontinue Support for Windows 10

Microsoft is cautioning users that on **October 14, 2025**, Windows will no longer provide free software updates for Windows 10. This means elevated security risk for everyone using this operating system, unless they:

1. **Upgrade to Windows 11**, or
2. **Pay for Extended Security Updates (ESU)** through Microsoft's program, which will support updates through October 13, 2026

The ESU option essentially buys users another year before they will be forced to upgrade regardless.

## What You Can Do

### Option 1: Upgrade to Windows 11

Not all devices will support Windows 11, but if you're running Windows 10 and your laptop supports the newest version, [now's the time to upgrade](https://www.microsoft.com/en-us/software-download/windows11).

### Option 2: Switch to Linux

Your device can likely run other operating systems. With many versions of Linux available, getting started can feel intimidating, but there are meaningful advantages to free and open source software:

- **Cost**: It's free
- **Control**: You control how it works
- **Transferable skills**: Much of what you learn on one Linux distribution transfers to others

Don't get too hung up on choosing the "perfect" distribution—getting started somewhere is most important. [Check out this guide](https://www.zdnet.com/article/how-to-choose-the-right-linux-desktop-distribution-for-you/) to learn more about choosing a Linux distribution.

### Option 3: Install ChromeOS Flex

If you're comfortable with Google and want a user-friendly experience, ChromeOS Flex is a decent free alternative to Windows 10 that you can install on older hardware.

**Advantages of ChromeOS Flex:**
- Extends the life of aging hardware
- Promises [a decade of updates for recent devices](https://support.google.com/chrome/a/answer/6220366?hl=en)
- Provides a relatively safe environment for opening suspicious files
- Works well as a travel computer that can be [easily wiped](https://support.google.com/chromebook/answer/183084?hl=en)

[Check out Wired's guide](https://www.wired.com/story/how-to-install-chromeos-flex/) to learn how to set it up.
