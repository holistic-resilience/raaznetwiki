# Privacy Guides Privacy Policy

## What is Privacy Guides?

Privacy Guides is a community project operated by active contributors. The public list of team members [can be found on the forum](https://discuss.privacyguides.net/u?group=team&order=solutions&period=all). Privacy Guides is legally administered by [MAGIC Grants](https://magicgrants.org/), a 501(c)(3) public charity acting as the fiscal host.

### Covered Services

This privacy notice covers all Privacy Guides projects authorized and operated by the MAGIC Privacy Guides Fund executive committee:

- [privacyguides.org](https://www.privacyguides.org/) - Main website
- [discuss.privacyguides.net](https://discuss.privacyguides.net/) - Community forum
- [code.privacyguides.dev](https://code.privacyguides.dev/) - Public source code repositories

### Third-Party Services (Separate Policies)

- **Donations**: [donate.magicgrants.org](https://donate.magicgrants.org/privacyguides) - See [MAGIC Grants privacy policy](https://donate.magicgrants.org/privacy)
- **Merchandise**: [shop.privacyguides.org](https://shop.privacyguides.org/) - See [Fourthwall privacy policy](https://shop.privacyguides.org/pages/privacy-policy)

## How Data is Collected

Privacy Guides collects data when you:

- Visit websites
- Create and use an account
- Post, send private messages, or participate in community discussions
- Sign up for mailing lists, email notifications, or announcements
- Become a donating member
- Donate via GitHub Sponsors
- Link MAGIC Grants account to forum account
- Contribute to open-source projects
- Contact Privacy Guides

> **Note**: Privacy Guides does not buy or receive data from data brokers.

---

## Data Sales and Sharing

**Privacy Guides does not sell personal information.** Additionally, Privacy Guides does not share personal information with third parties for advertising purposes.

---

## Types of Personal Information Collected

### Website Visit Information

When visiting Privacy Guides websites, server logs and other methods collect data about pages visited and timing. This data is used to:

- Optimize website performance
- Diagnose and debug technical errors
- Defend against abuse and technical attacks
- Compile page/topic popularity statistics
- Compile browser and device statistics

**Retention**: Generally stored for a few weeks. Extended retention may occur during technical attacks or investigations. Aggregated statistics (non-personally identifiable) are stored indefinitely.

### Account Data

Account creation requires:
- Email address
- Username
- Password

This data is used to:
- Identify users on the website
- Create personalized pages (e.g., profile page)
- Send notifications about posts and activity
- Reset passwords and maintain account security
- Contact users about account-related matters or legal requests

**Optional profile information** (name, biography, location, birthday) can be provided in profile settings. This information is available to others with forum access and can be erased at any time.

**Retention**: Stored as long as the account remains open.

### Posts and Activity Data

Privacy Guides collects:
- Content of posts
- Bookmarks, likes, and links followed
- Private messages (available to senders, recipients, moderators, and administrators)

**Retention**: Stored as long as the account remains open.

### Mailing List Data

When signing up for mailing lists or announcements, Privacy Guides collects form information including email address.

**Retention**: Stored until unsubscription.

### Sensitive Personal Information

Privacy Guides does **not** intentionally collect sensitive personal information, including:
- Government identification numbers
- Racial or ethnic origin information
- Political opinions
- Genetic or biometric data
- Health data
- GDPR special categories of personal data

### Open Source Contributor Data

Contributors may be asked to provide identifying and contact information (name, email address) to maintain website, software, and license agreement integrity.

**Retention**: Stored as long as contributions are incorporated into open source software.

### Donation Data

#### Membership on privacyguides.org
- **Collected**: Email address, optional name
- **Payment processing**: Handled by Stripe
- **Forum linking**: Membership status may be linked to forum account for perks
- **Public display**: Username and profile picture may be shared publicly if Member title/flair is enabled (revocable in profile settings)

#### GitHub Sponsors
- **Collected**: GitHub username and profile picture
- **Payment processing**: Handled by GitHub and Stripe
- **Public display**: If donation is made public, username and profile picture are shared on websites (revocable at any time)

#### donate.magicgrants.org
- Separate privacy policy applies
- Information processed to grant forum perks
- Public display optional via member title/flair settings

**Retention**: Stored while membership is active or until accounts are unlinked.

### Contact Data

When contacting Privacy Guides via email, Signal, or other methods:
- Information submitted is collected
- Identifying information associated with the contact account is collected

**Retention**: Stored during contact period and up to 1 year after last contact. Deletion requests are typically processed within 7 days.

### Local Storage and Session Storage

**Local Storage**: Stores color scheme preference (client-side only)

**Session Storage**: Caches website version number and GitHub repository stars/forks (fetched once per session)

## GDPR Compliance

Privacy Guides respects rights under the European Union's General Data Protection Regulation (GDPR).

### GDPR Rights

| Right | Description |
|-------|-------------|
| Access | Access your personal data |
| Rectification | Correct inaccurate or incomplete data |
| Erasure | Delete your personal data |
| Data Portability | Receive data in portable format |
| Restrict Processing | Limit data processing |
| Object | Object to automated decision-making and direct marketing |
| Lodge Complaint | File with supervisory authority |

### Lawful Basis for Processing

| Basis | Description |
|-------|-------------|
| **Consent** | Clear consent given for processing (withdrawable via account settings) |
| **Contract** | Data provided to participate in services per terms of service |
| **Legitimate Interest** | Processing for fraud prevention, security, or third-party legitimate interests |

### Detailed Lawful Basis by Activity

| Activity | Data Collected | Lawful Basis |

### International Data Transfers

- Personal data processed on servers outside the European Union
- Subprocessors have personnel and computers outside the EU
- Personnel in USA, Australia, and other non-EU countries without adequacy decisions
- Likely subject to FISA Section 702

**Safeguards**:
- No orders or requests received under FISA 702 or similar laws
- No court orders preventing disclosure of such requests
- Policy in place for responding to such orders: suspend processing, notify affected users, minimize disclosure, resist disclosure

## Automated Decisions

### Spam Classification
Posts are automatically classified as spam based on analysis of posts across forums. Wrongful blocks can be appealed to forum moderators.

### Trust Levels
Post and activity data determines badges and trust levels, which affect:
- Participation capabilities (e.g., image uploads)
- Access to moderation and management powers

Trust level adjustments can be requested from forum administrators.

### Community Flags
Posts may be automatically hidden or posting suspended based on user flags. Moderators review and can override these decisions.

## Inactive Account Deletion

Accounts with no public activity are deleted after **3 years** of inactivity. Accounts with public activity are retained to continue publishing activity per original request.

Data deletion can be requested at any time regardless of this policy.
