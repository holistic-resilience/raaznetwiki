# Tutorial on Disabling Tracking of Viewed Links in Facebook

## What is Link History?

If you have recently checked your Facebook settings, you have likely encountered the "Link History" option. This new feature is a library that saves all the links you click on.

Meta has introduced this option as a "feature" to help users not lose any links. But in reality, this is a new method for tracking user behavior.

> **Note:** This feature has been automatically enabled on user accounts.

## Disabling Steps

![Tutorial on disabling storage and tracking of viewed links in Facebook](https://certfa.com/res/get/i/u/f75e30dd-b741-46ed-ba4b-e04d1eed289c.jpg)

1. Open the Facebook app
2. Go to the **Menu** section
3. Click on **Settings & privacy**
4. Select the **Settings** option
5. Find **Browser** on this page
6. Disable the **Link History** option
